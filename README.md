# aiLern - AI-Powered Learning Platform

A comprehensive learning platform that leverages AI to provide personalized education experiences.

## Features

- AI-powered personalized learning paths
- Interactive courses and lessons
- Progress tracking and assessments
- Community features
- Multi-language support

## Tech Stack

### Backend
- Node.js
- Express.js
- MongoDB/Database integration

### Frontend
- React
- Vite
- Tailwind CSS

## Getting Started

### Backend Setup
```bash
cd backend
npm install
cp .env.example .env
# Configure your environment variables
npm start
```

### Frontend Setup
```bash
cd frontend
npm install
cp .env.example .env
# Configure your environment variables
npm run dev
```

## Project Structure

- `backend/` - Server-side application
- `frontend/` - Client-side React application
- `docs/` - Project documentation
